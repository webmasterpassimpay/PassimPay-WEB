Модуль оплаты для PrestaShop
Готовый плагин от PassimPay для CMS PrestaShop позволяет легко и быстро интегрировать прием онлайн-платежей на ваш сайт.
Чтобы начать принимать оплаты, достаточно выполнить несколько шагов:
1. Зарегистрироваться в PassimPay,
2. Скачать плагин,
3. Установить модуль для PrestaShop на своем сайте,
4. Настроить все необходимые методы оплаты и внешний вид платежной страницы.
Модуль оплаты PrestaShop позволяет вам подключить и предложить клиентам любой удобный способ оплаты криптовалютой: Bitcoin, Ethereum, Bitcoin Cash, Dogecoin, USD Coin, Tether и множество других.
Вы сможете принимать платежи из стран Европы, Азии, Северной и Латинской Америки, СНГ.

Чтобы начать принимать оплаты, достаточно выполнить несколько шагов

1. Зарегистрируйтесь на сайте PassimPay

2. Добавьте ваш проект в личном кабинете

3. Скачайте плагин для PrestaShop

4. Проведите интеграцию по инструкции

5. Начать получать платежи

После установки и настройки плагина все оплаты на вашем сайте будут проходить через платежную страницу PassimPay.
Выбрав товар, пользователь нажмет кнопку «Оплатить», попадет на страницу оплаты, введет реквизиты выбранной криптовалты и подтвердит платеж.
Вся информация о платежах будет отображаться в вашем личном кабинете. Там можно отслеживать дату и время платежа, его статус, а также выполнить возврат или вывести средства.
Как работает модуль оплаты PrestaShop
Подключив платежный модуль PrestaShop, вы получаете все преимущества работы с PassimPay. Вот самые главные из них:
  • Больше 50 валют для приема платежей, чтобы расширить границы вашего бизнеса;
  • Подробная аналитика платежей в личном кабинете и отслеживание статуса транзакции в режиме реального времени;
  • Служба поддержки, работающая 24/7 для вашего бизнеса и плательщиков;
  • Возможность настроить необходимые поля и внешний вид страницы под стиль бренда;
  • Гибкий антифрод и 200+ правил проверки транзакций на предмет мошенничества;
Если вам нужна помощь с подключением модуля, вы можете обратиться к нашей команде технической поддержки.
Инструкция по подключению
  • 1.Зарегистрируйтесь на сайте https://passimpay.io/register
  • 2.Ознакомьтесь с инструкциями в личном кабинете;
  • 3.Добавьте ваш проект в личном кабинете, заполнив необходимые поля;
  • 1.При добавлении проекта проверьте корректность указанных ссылок.
    • Успешный URL:
      https://domain.com/index.php?route=extension/payment/passimpay/response
    • Неудачный URL:
      https://domain.com/index.php?route=extension/payment/passimpay/response&fail=1
    • URL для уведомлений:
      https://domain.com/index.php?route=payment/passimpay/callback
  • 2.Скопируйте содержимое папки admin и catalog в корень установкиOpenCart (при вставке файлы необходимо объединить);
  • 3.В админ-панеле OpenCart переходим в раздел «Дополнения» «Оплата» активируеми настраиваем способ оплаты «PassimPay».
  • 4.Заполните платежные реквизиты в кошельке
    • ApiKey:
      Для получения ApiKey перейдите в раздел «Мои проекты» в личном кабинете.
    • ShopID:
      Для получения ShopID необходимо добавить проект в личном кабинете. ID состоит из 16 символов и уникален для каждого магазина.
    • Заголовок:
      Название способа оплаты (можно указать «Crypto (BTC/LTC/ETH/USDT) - CryptoCloud»)
    • Статус заказа после оплаты:
      Определяет статус заказа после успешной оплаты
    • Статус:
      Включает и отключает способ оплаты
    • Порядок сортировки:
      Определяет порядок отображения метода оплаты (можно указать “1”)
